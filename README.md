# OPC UA сервер

Сбор данных с устройств, поддерживающих протокол MODBUS (RTU, TCP) и информационный обмен с ПО верхнего уровня (напр. SCADA) по протоколу OPC UA.

    
* Импорт параметров из json-файла.
* Поддержка протоколов ModBus TCP и ModBus RTU.
* Стек OPC UA реализован на основе библиотеки open62541 (https://github.com/open62541/open62541).
